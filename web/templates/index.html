Guess IPL match results from the teams you pick

You enter the two squads (11 players each) and a few match details. The tool estimates who might win and standout players. After the real match, you can log what actually happened and see side‑by‑side if you were close.

Start a prediction See predictions vs real results

How to use this (3 steps)

  1. Predict a match — Choose teams, paste both XIs, say who bats first if you know it, then run the model.
  2. Save the output — Copy the JSON or use your usual workflow to append a row to the prediction log.
  3. Add the real result — When the game is over, fill in winner, top scorer, POTM, etc. Then open Compare to results to see the table.

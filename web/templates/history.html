How to read this

  • Read down each card: left column is the question, middle is the model before the game, right is what you added after.
  • Rough win split — Only the model fills this (e.g. SH 39% · RCB 61%). Left team in the title = first number. Not a betting price.
  • Match? — Own column for Match / Miss so names are not glued to the tag.
  • Top 5 / Top 3 — After column uses your bbb_match_file ball-by-ball CSV: top 5 batters by runs, top 3 bowlers by wickets. Match = model list and actual list are the same in the same order (IDs compared after normalizing). No file or short lists → —.
  • Top runs / wickets (pick) — Single “best” picks: Match if model best bat/bowl equals the logged top scorer / top wicket-taker (same person).
